# Bhavanam Likith Reddy Portfolio

This is a dependency-free static portfolio site. It can be deployed by uploading this folder to any static host.

## Deploy Options

### Netlify

Drag and drop the `deployable-portfolio` folder into Netlify, or connect a GitHub repository that contains this folder.

### Vercel

Import the repository in Vercel and set the project root to `deployable-portfolio`.

### GitHub Pages

Push the contents of this folder to a repository and enable GitHub Pages from the repository settings.

## Files

- `index.html` - page content
- `styles.css` - visual design and responsive layout
- `script.js` - navigation, reveal animation, project toggles, and counters
- `assets/` - resume, favicon, and social preview image
